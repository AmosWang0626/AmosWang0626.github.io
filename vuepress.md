## Quick Start

```
# install
yarn install

# start
yarn dev

# build
yarn build
```

**yarn 和 npm 区别**
1. 速度快。yarn install的时候是并行执行的；npm是串行的。
2. 输出更简洁。
3. 语义化更清晰。yarn改install为add,uninstall为remove等。
